# ZX-AE v1.0

A lightweight authenticated encrypted container format based on XChaCha20-Poly1305.

## Features
- XChaCha20-Poly1305 AEAD
- Strict header format
- Compression profiles (zstd)
- Deterministic nonce handling
- Clean Rust reference implementation

## Directories
- `src/` – core library
- `cli/` – ZX-AE command-line tool
- `spec/` – published format documentation
- `examples/` – small usage demos
- `tests/` – integration tests
